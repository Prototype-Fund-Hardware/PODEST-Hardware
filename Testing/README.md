# PODEST-Hardware

This is the repository for the [PODEST](https://okfn.de/projekte/podest/) project, which is being developed in collaboration with TU Berlin, BTU Cottbus, and OKF DE.. The goal of the project is to improve and develop open-source hardware for private power generation. The following components are planned:
- BMS (48V DC input voltage)
- Inverter
- Small wind turbine
- Control system
